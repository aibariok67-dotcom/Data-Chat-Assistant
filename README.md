# 📊 Data Chat Assistant

AI-ассистент для анализа данных на естественном языке. Загружаешь CSV — задаёшь вопросы — получаешь ответы с кодом и графиками.

## Стек
- **Python** + **pandas** — обработка данных
- **OpenAI GPT-4o-mini** — генерация кода и ответов
- **Streamlit** — веб-интерфейс
- **Matplotlib** — визуализация

## Быстрый старт

```bash
# 1. Клонируй репозиторий
git clone https://github.com/YOUR_USERNAME/data-chat-assistant
cd data-chat-assistant

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Запусти
streamlit run app.py
```

## Как использовать

1. Введи OpenAI API Key в боковой панели
2. Загрузи любой CSV файл (или используй `sample_data.csv` для демо)
3. Задавай вопросы на русском или английском:
   - "Какой месяц был самым прибыльным?"
   - "Построй гистограмму продаж по категориям"
   - "Есть ли пропуски в данных?"
   - "Покажи топ-5 менеджеров по продажам"

## Примеры запросов

| Вопрос | Что делает |
|--------|-----------|
| Какой месяц самый прибыльный? | groupby + sum + idxmax |
| Покажи топ продуктов | sort_values + head |
| Есть ли пропуски? | isnull().sum() |
| Построй график продаж | matplotlib bar chart |
| Корреляция колонок | df.corr() + heatmap |
