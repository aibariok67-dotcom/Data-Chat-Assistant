# Data Chat Assistant

Загружаешь CSV — задаёшь вопросы на русском — получаешь ответы, таблицы и графики.

Под капотом: GPT пишет pandas-код, код выполняется, результат показывается в чате.

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![Streamlit](https://img.shields.io/badge/Streamlit-1.32-red) ![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4o--mini-green)

---

## Что умеет

- Отвечать на вопросы о данных: *"Какой месяц был самым прибыльным?"*
- Строить графики: *"Построй гистограмму продаж"*
- Сравнивать: *"Сравни продажи по регионам"*
- Проверять качество данных: *"Есть ли пропуски?"*
- Показывать топы: *"Покажи топ-5 товаров"*

---

## Запуск

```bash
# 1. Установи зависимости
pip install streamlit pandas openai matplotlib

# 2. Запусти
streamlit run app.py
```

Открой http://localhost:8501, введи OpenAI API ключ и загрузи любой CSV.

Для быстрого теста используй `sample_data.csv` — это продажи за 2024 год (7 продуктов, 4 региона, 82 записи).
