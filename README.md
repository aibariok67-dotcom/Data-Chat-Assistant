# Data Chat Assistant

**Программа, которая помогает разбираться в таблицах через обычный разговор.**  
Вы загружаете файл с данными (таблицу в формате CSV — как «Сохранить как CSV» из Excel или Google Таблиц) и пишете вопросы по-русски. Ответ приходит в виде текста, цифр, таблиц или графиков — как удобнее для вопроса.

*По смыслу это похоже на умного помощника по вашей таблице: он «смотрит» на данные и отвечает, опираясь на них.*

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![Streamlit](https://img.shields.io/badge/Streamlit-1.32-red) ![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4o--mini-green)

---

## Кому это может пригодиться

- Нужно быстро понять продажи, расходы, рейтинги — без формул в Excel.
- Хочется увидеть график, не разбираясь в настройках диаграмм.
- Нужны простые ответы: «кто лидер», «где просадка», «есть ли ошибки в данных».

---

## Что можно спросить (примеры)

- *«В каком месяце была самая большая прибыль?»*
- *«Покажи продажи по регионам на графике»*
- *«Сравни, как вели себя разные категории»*
- *«Есть ли в таблице пустые ячейки или странные значения?»*
- *«Покажи пять товаров с самыми большими продажами»*

## Что понадобится перед запуском

1. **Компьютер** с установленным **Python** (версия 3.10 или новее). Python — это бесплатная среда, в которой работает программа; её обычно ставят с сайта [python.org](https://www.python.org/downloads/).
2. **Доступ в интернет** — чтобы обращаться к сервису искусственного интеллекта (OpenAI).
3. **Ключ API OpenAI** — это как логин/пароль к платному сервису OpenAI: его вводят в окне программы после запуска. Ключ выдаётся в личном кабинете на сайте OpenAI после регистрации и пополнения баланса (условия и цены — на их сайте).

Без ключа программа не сможет отвечать на вопросы о ваших данных.

## Как скачать проект на компьютер

Репозиторий на GitHub: [aibariok67/Data-Chat-Assistant](https://github.com/aibariok67/Data-Chat-Assistant).

**Через Git (клонирование)** — нужна бесплатная программа [Git](https://git-scm.com/downloads). В терминале:

```bash
git clone https://github.com/aibariok67/Data-Chat-Assistant.git
cd Data-Chat-Assistant
```

После `cd` вы должны оказаться в папке, где лежит файл `app.py`. Если переименуете репозиторий на GitHub, ссылку и имя папки после `cd` возьмите из зелёной кнопки **Code** → HTTPS на странице репозитория.

**Без Git:** на [странице репозитория](https://github.com/aibariok67/Data-Chat-Assistant) **Code** → **Download ZIP**, распакуйте архив и дальше работайте из **этой** папки в терминале (там же должен быть `app.py`).

## Как запустить на своём компьютере

Откройте терминал (в Windows — «Командная строка» или PowerShell, на Mac — «Терминал») **в папке проекта** (где лежит `app.py`) и выполните по очереди:

```bash
pip install -r requirements.txt
streamlit run app.py
```

После запуска браузер обычно откроется сам. Если нет — в терминале будет адрес вида **http://localhost:8501** — откройте его вручную.

Дальше: введите ключ API, загрузите CSV и задайте вопрос.

**Попробовать на готовом примере:** в комплекте есть файл `sample_data.csv` — вымышленные продажи за 2024 год (несколько продуктов и регионов). Загрузите его и поэкспериментируйте с вопросами.

## Для тех, кто любит технические детали

| Что используется | Зачем |
|------------------|--------|
| **Streamlit** | Простой веб-интерфейс в браузере |
| **pandas** | Работа с таблицами и расчёты |
| **OpenAI (модель GPT)** | Понимание вопроса и подготовка шагов анализа |
| **matplotlib** | Построение графиков |

Внутри модель предлагает код на Python для вашей таблицы; код выполняется локально, в интерфейс попадают уже готовые ответы и картинки.
